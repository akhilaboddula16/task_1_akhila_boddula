# LinkedIn Post

Built a `Luxury Travel Consultant AI` using `Python`, `Streamlit`, and the `Groq API` as part of a prompt engineering and persona design challenge.

The core idea behind this project was not just to build a chatbot, but to design an AI assistant that behaves like a professional `Luxury Travel Consultant` for a high-end travel agency.

### Project Scenario

A premium travel agency wants to automate customer inquiries with an AI assistant that:

- responds in a refined and professional tone
- stays within a luxury travel domain
- knows when to offer discounts
- suggests value-added benefits when discounts are not allowed
- never mentions competitors
- handles difficult customer interactions without breaking character

### What I focused on

- `System Prompt Architecture` to define a strict persona, rules, tone, and response boundaries
- `Persona Engineering` so the assistant consistently behaves like a luxury travel advisor
- `Few-Shot Prompting` to guide the model with examples of ideal responses
- `LLM Integration` using the Groq API for fast conversational responses
- `Streamlit UI Design` to present the assistant through a polished, app-like interface
- `Response Structuring` so answers are more readable, point-wise, and user-friendly

### Outcome

The result is an AI travel assistant that can manage premium travel conversations with a controlled tone, clear response style, and better business alignment. This project helped me understand how system prompting can shape model behavior far beyond simple question answering.

It was a great hands-on exercise in combining:

- prompt engineering
- persona design
- UI development
- real-world AI application thinking
